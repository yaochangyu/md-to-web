# 語法範例

## 嵌入（Embed）

將另一份 `.md` 的完整內容展開在當前位置：

```markdown
![[./path/to/file.md]]
```

build 時會讀取目標檔案並直接插入，只展開一層（不支援巢狀嵌入）。

## 下載按鈕

非圖片附件自動渲染成下載按鈕：

```markdown
![按鈕文字](./assets/doc/example.pdf)
```

圖片副檔名（`.png .jpg .jpeg .gif .svg .webp .avif .bmp .ico`）照常顯示，其餘一律轉為下載按鈕。

## 同頁標題跳轉

```markdown
[[#標題文字]]
```

- **Obsidian**：原生支援，點擊直接跳轉
- **VitePress**：build 時自動轉成 `[標題文字](#slug)`

## 附件目錄慣例

```
assets/
└── {文件名稱}/
    ├── *.png / *.jpg        ← 圖片，正常渲染
    ├── *.md                 ← 用 ![[path]] 嵌入 或 ![label](path) 下載
    └── *.pdf / *.zip / ...  ← 用 ![label](path) 下載
```

## 圖片燈箱

點擊任何圖片即可放大，再點一下或按 `Esc` 關閉。

## 標題收合

點擊標題前的 `▼` 符號可摺疊/展開該區段。

## 右側目錄

- 關鍵字搜尋：輸入後即時篩選目錄項目
- 全部展開/收闔：點擊 `↕` 按鈕
- 自動捲動同步：點擊 `🎯` 按鈕切換

## 側欄收合

- 左側選單：點擊邊緣的 `❮` / `❯` 按鈕
- 右側目錄：點擊邊緣的 `❯` / `❮` 按鈕
- 兩側皆可拖拉調整寬度（寬度自動記憶）
